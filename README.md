# xml-meteo
Ce projet contient un **document XML** qui présente les **mesures de températures** des différentes villes à une date donnée.  
Chaque mesure est qualifiée par une **date** et contient un **ensemble de villes**.  
Chaque ville est qualifiée par un **nom** et une **température**.

Le but est de créer un **document XML valide** basé sur un **schéma XSD**, et de reformater les données pour les afficher dans un **graphe**.
